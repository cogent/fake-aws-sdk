# FakeAWS

A fake implementation of the AWS Ruby SDK, for testing.

## Installation

Add this line to your application's Gemfile:

    gem 'fake-aws-sdk'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fake-aws-sdk
